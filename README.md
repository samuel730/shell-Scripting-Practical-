# shell-Scripting-Practical-
## Introduction

In system administration and DevOps, automation plays a key role in improving efficiency, reducing human error, and ensuring consistency in repetitive tasks. One of the most common administrative tasks is data backup, which helps protect important files and system data from accidental loss or system failure.

This project demonstrates how to use **Bash shell scripting** in a Linux environment to automate the backup of a directory. The script compresses files into a timestamped archive and stores them in a designated backup location. It also introduces task scheduling using **cron jobs**, enabling the backup process to run automatically at scheduled intervals without manual intervention.

Through this project, fundamental DevOps skills such as scripting, automation, file system management, and task scheduling are applied in a practical scenario.
